addEventListener — 이벤트 듣기

버튼 클릭, 키 입력 등 사용자 행동에 반응하는 방법

addEventListener는 HTML 요소에 "이런 일이 생기면 이 함수를 실행해!"라고 등록하는 것입니다.
버튼 클릭, 키보드 입력, 마우스 이동 등 모든 사용자 행동을 감지할 수 있습니다.
// 버튼 클릭 이벤트 const btn = document.getElementById('saveBtn'); btn.addEventListener('click', saveMemo); // 키보드 이벤트 (Enter 키) input.addEventListener('keydown', function(e) { if (e.key === 'Enter') { saveMemo(); } }); // 함수 정의 function saveMemo() { // 메모 저장 처리... }
📌

자주 쓰는 이벤트

'click' 클릭할 때
'keydown' 키를 누를 때
'input' 값이 바뀔 때
'submit' 폼 제출할 때

localStorage — 브라우저 저장소

새로고침해도 사라지지 않는 데이터 저장 공간

기본 사용법

JS // 문자열 저장 (key, value) localStorage.setItem('name', '홍길동'); // 문자열 불러오기 const name = localStorage.getItem('name'); // 삭제 localStorage.removeItem('name'); // 배열/객체 저장 (JSON 변환 필요) const memos = ['메모1', '메모2']; localStorage.setItem('memos', JSON.stringify(memos)); // 배열 불러오기 const loaded = JSON.parse( localStorage.getItem('memos') || '[]' );

메모 저장 전체 흐름

JS // 1) 기존 메모 불러오기 let memos = JSON.parse( localStorage.getItem('memos') || '[]' ); // 2) 새 메모 추가 memos.push({ text: input.value, time: new Date().toLocaleString() }); // 3) 다시 저장 localStorage.setItem('memos', JSON.stringify(memos)); // 4) 화면 새로 그리기 renderMemos();
🔄

JSON — 데이터를 문자열로 변환

JavaScript 객체를 저장 가능한 텍스트로 바꾸는 방법

localStorage는 오직 문자열만 저장할 수 있습니다. 그래서 배열/객체를 저장할 때는 JSON.stringify()로 변환하고, 꺼낼 때는 JSON.parse()로 복원합니다.
// JS 배열/객체 → 문자열로 const arr = [1, 2, 3]; JSON.stringify(arr); // → '[1,2,3]' (문자열) const obj = { name: '홍길동', age: 30 }; JSON.stringify(obj); // → '{"name":"홍길동","age":30}' // 문자열 → JS 배열/객체로 JSON.parse('[1,2,3]'); // → [1, 2, 3] (진짜 배열)
⚠️

null 처리 패턴

처음 실행할 때는 localStorage에 데이터가 없어서 null을 반환합니다.

const data = JSON.parse(
  localStorage.getItem('memos')
  || '[]' // null이면 빈 배열

자주 막히는 문제 & 해결법

여기서 멈추는 게 정상입니다 — 옆 사람도 같은 곳에서 막혀 있어요

🔍 "Go Live 버튼이 안 보여요"

VS Code 왼쪽 확장(Extensions) 메뉴에서 Live Server를 검색해 설치한 뒤, VS Code를 껐다가 다시 켜세요.

💾 "고쳤는데 화면이 그대로예요"

저장(Ctrl+S)이 먼저입니다! 저장하지 않으면 반영되지 않아요. 저장 후 브라우저에서 새로고침(F5) 하세요.

⏸️ "Cline이 멈춰서 아무것도 안 해요"

파일을 만들거나 고치기 전에 승인(Approve)을 기다리는 중입니다. Cline 채팅창의 승인 버튼을 눌러주세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.

JS function saveMemo() { const text = input.value.trim(); if (!text) return; // 빈 내용 차단 const memos = loadMemos(); memos.unshift({ // 맨 앞에 추가 id: Date.now(), text: text, time: new Date().toLocaleString('ko-KR') }); localStorage.setItem('memos', JSON.stringify(memos)); input.value = ''; // 입력창 초기화 renderMemos(); }

메모 렌더링 함수

JS function renderMemos() { const memos = loadMemos(); const list = document.getElementById('list'); list.innerHTML = memos.map(memo => `<div class="memo"> <p>${memo.text}</p> <small>${memo.time}</small> <button onclick="deleteMemo(${memo.id})"> ✕ </button> </div>` ).join(''); }

템플릿 리터럴 — 문자열 조합

백틱(`)으로 변수를 자연스럽게 문자열에 넣는 방법

템플릿 리터럴은 작은따옴표/큰따옴표 대신 백틱(` `)을 사용해서
${변수명} 형식으로 변수 값을 문자열 안에 자연스럽게 넣습니다.
// ❌ 예전 방식 (+ 로 이어붙이기) const name = '홍길동'; const msg = '안녕하세요, ' + name + '님! ' + '좋은 하루 되세요.'; // ✅ 템플릿 리터럴 (훨씬 깔끔!) const msg2 = `안녕하세요, ${name}님! 좋은 하루 되세요.`; // 계산식도 넣을 수 있음 const a = 3, b = 5; console.log(`합계: ${a + b}원`); // → "합계: 8원"
💡

백틱(`)은 어디에?

키보드 숫자 1 왼쪽, Esc 아래

한/영 전환 상태에서 눌러야 합니다.
한글 상태에서는 ₩ (원화 기호)가 입력됩니다.

innerHTML & select 값 읽기

드롭다운 선택값을 읽고 카드에 적용하기

select 값 읽기

HTML+JS <!-- HTML --> <select id="situation"> <option value="birthday">생일</option> <option value="thanks">감사</option> <option value="newyear">신년</option> </select> // JavaScript - 선택된 값 읽기 const sel = document .getElementById('situation'); const val = sel.value; // → 'birthday' 또는 'thanks'...

상황별 메시지 분기

JS function getMsg(name, situation) { if (situation === 'birthday') { return `🎂 ${name}님, 생일을 진심으로 축하합니다! 오늘 하루 행복하세요!`; } else if (situation === 'thanks') { return `🙏 ${name}님, 항상 도움 주셔서 감사합니다!`; } else { return `🎉 ${name}님, 새해 복 많이 받으세요!`; } }

클립보드 복사 기능

생성된 메시지를 버튼 한 번으로 복사

// 최신 방식 (navigator.clipboard) async function copyToClipboard() { const text = document .getElementById('result') .textContent; await navigator.clipboard .writeText(text); // 복사 완료 피드백 copyBtn.textContent = '✅ 복사됨!'; setTimeout(() => { copyBtn.textContent = '📋 복사하기'; }, 2000); }
🔄

async / await 란?

클립보드 복사는 비동기 작업입니다. async/await는 완료를 기다리는 문법입니다.

지금은 "복사할 때 이렇게 쓴다"만 기억하면 충분합니다!

🆘

자주 막히는 문제 & 해결법

여기서 멈추는 게 정상입니다 — 옆 사람도 같은 곳에서 막혀 있어요

🔍 "Go Live 버튼이 안 보여요"

VS Code 왼쪽 확장(Extensions) 메뉴에서 Live Server를 검색해 설치한 뒤, VS Code를 껐다가 다시 켜세요.

💾 "고쳤는데 화면이 그대로예요"

저장(Ctrl+S)이 먼저입니다! 저장하지 않으면 반영되지 않아요. 저장 후 브라우저에서 새로고침(F5) 하세요.

⏸️ "Cline이 멈춰서 아무것도 안 해요"

파일을 만들거나 고치기 전에 승인(Approve)을 기다리는 중입니다. Cline 채팅창의 승인 버튼을 눌러주세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.
🔍

코드 핵심 — 카드 생성 함수

입력값 읽기 → 메시지 조합 → 화면 출력

JS function generateCard() { // 1) 입력값 읽기 const name = document.getElementById('name').value.trim(); const situation = document.getElementById('situation').value; const theme = document.getElementById('theme').value; // 2) 유효성 검사 if (!name) { alert('이름을 입력해주세요!'); return; } // 3) 메시지 생성 (템플릿 리터럴) const message = getMsg(name, situation); // 4) 카드에 출력 + 테마 적용 const card = document.getElementById('card'); card.innerHTML = `<p>${message}</p>`; card.className = `card theme-${theme}`; card.style.display = 'block'; }

수정 미션, 이렇게 진행하세요

바이브 코딩의 진짜 실력은 "고쳐달라고 잘 말하기"에서 나옵니다

1
같은 Cline 대화에서 이어가기
새 작업(New Task)을 열지 마세요! 코드를 만들어 준 대화에 이어서 요청해야 Cline이 맥락을 기억합니다.
2
미션 문구 붙여넣기
다음 슬라이드의 미션 프롬프트를 복사하거나, 원하는 변화를 말로 설명하세요. 문법·전문용어는 몰라도 됩니다.
3
변경 내용 확인하고 승인
Cline이 파일을 직접 고쳐줍니다. 어떤 부분이 바뀌는지(diff) 훑어보고 승인하세요. 직접 타이핑할 필요가 없어요.
4
저장하고 확인
저장(Ctrl+S) → 브라우저 새로고침(F5) → 바뀐 부분을 확인합니다.

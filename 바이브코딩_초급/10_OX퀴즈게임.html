객체 배열 — 퀴즈 데이터 관리

여러 속성을 가진 데이터를 배열로 관리하는 방법

객체는 여러 관련 정보를 하나로 묶는 방법입니다. 객체 배열은 여러 객체를 배열로 모아서 퀴즈 문제처럼 비슷한 구조의 데이터를 관리합니다.
// 퀴즈 데이터 (객체 배열) const quizData = [ { question: 'HTML은 프로그래밍 언어이다', answer: false, // X (마크업 언어) explanation: 'HTML은 마크업 언어입니다.' }, { question: 'CSS는 색상과 레이아웃을 담당한다', answer: true, // O explanation: 'CSS는 스타일을 담당합니다.' }, { question: 'JavaScript는 서버에서만 실행된다', answer: false, // X explanation: '브라우저에서도 실행됩니다.' } ]; // 접근 방법 quizData[0].question // 첫 번째 문제 quizData[0].answer // false (X) quizData.length // 3 (총 문제 수)
🆚

배열 vs 객체 vs 객체 배열

[] 배열: 순서 있는 목록
['홍길동', '김철수']

{} 객체: 키-값 쌍
{name:'홍', age:30}

[{}] 객체 배열: 둘의 조합
[{name:'홍',score:90},...]

화면 전환 패턴 — showScreen()

여러 화면(시작/퀴즈/결과)을 하나의 HTML로 관리

HTML+JS <!-- HTML: 3개 화면, 하나만 표시 --> <div id="startScreen">시작</div> <div id="quizScreen" class="hidden">퀴즈</div> <div id="resultScreen" class="hidden">결과</div> /* CSS */ .hidden { display: none; } // JavaScript function showScreen(screenId) { // 모든 화면 숨기기 ['startScreen','quizScreen','resultScreen'] .forEach(id => { document.getElementById(id) .classList.add('hidden'); }); // 원하는 화면만 표시 document.getElementById(screenId) .classList.remove('hidden'); } // 사용 예시 showScreen('quizScreen');

퀴즈 상태 변수들

currentIndex
현재 몇 번째 문제인지 (0부터 시작)
score
맞은 개수 (정답 시 +1)
wrongList
틀린 문제 번호를 저장하는 배열
answered
현재 문제에 이미 답했는지 여부

function answer(userAnswer) { if (answered) return; // 이미 답했으면 무시 answered = true; const quiz = quizData[currentIndex]; const isCorrect = (userAnswer === quiz.answer); if (isCorrect) { score++; showFeedback('정답! 🎉', '#059669'); } else { wrongList.push(currentIndex); showFeedback('오답 😢 → ' + quiz.explanation, '#dc2626'); } // 1.5초 후 다음 문제로 setTimeout(() => { currentIndex++; if (currentIndex < quizData.length) { showQuestion(); } else { showScreen('resultScreen'); // 게임 끝! } }, 1500); }

자주 막히는 문제 & 해결법

여기서 멈추는 게 정상입니다 — 옆 사람도 같은 곳에서 막혀 있어요

🔍 "Go Live 버튼이 안 보여요"

VS Code 왼쪽 확장(Extensions) 메뉴에서 Live Server를 검색해 설치한 뒤, VS Code를 껐다가 다시 켜세요.

💾 "고쳤는데 화면이 그대로예요"

저장(Ctrl+S)이 먼저입니다! 저장하지 않으면 반영되지 않아요. 저장 후 브라우저에서 새로고침(F5) 하세요.

⏸️ "Cline이 멈춰서 아무것도 안 해요"

파일을 만들거나 고치기 전에 승인(Approve)을 기다리는 중입니다. Cline 채팅창의 승인 버튼을 눌러주세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.

코드 핵심 — 문제 표시 & 초기화

showQuestion() 함수와 게임 재시작 패턴

JS // 현재 문제 표시 function showQuestion() { answered = false; const quiz = quizData[currentIndex]; document.getElementById('questionNum').textContent = `${currentIndex+1} / ${quizData.length}`; document.getElementById('questionText').textContent = quiz.question; // 진행률 바 업데이트 const pct = ((currentIndex) / quizData.length) * 100; document.getElementById('progressBar').style.width = pct + '%'; } // 게임 재시작 function restart() { currentIndex = 0; // 첫 번째 문제로 score = 0; // 점수 초기화 wrongList = []; // 오답 목록 초기화 showScreen('quizScreen'); showQuestion(); }

1

2
미션 문구 붙여넣기
다음 슬라이드의 미션 프롬프트를 복사하거나, 원하는 변화를 말로 설명하세요. 문법·전문용어는 몰라도 됩니다.
3
변경 내용 확인하고 승인
Cline이 파일을 직접 고쳐줍니다. 어떤 부분이 바뀌는지(diff) 훑어보고 승인하세요. 직접 타이핑할 필요가 없어요.
4
저장하고 확인
저장(Ctrl+S) → 브라우저 새로고침(F5) → 바뀐 부분을 확인합니다.

Math.random() — 랜덤 숫자 생성

0 이상 1 미만의 랜덤 소수를 생성하는 내장 함수

Math.random()은 0 이상 1 미만 (0 ≤ x < 1) 의 랜덤 소수를 반환합니다.
배열의 인덱스와 조합하면 랜덤 항목 선택이 가능합니다.
// 기본: 0.0 ~ 0.999... 소수 Math.random(); // → 0.7341... // 0~9 정수 랜덤 Math.floor(Math.random() * 10); // 1~6 주사위 Math.floor(Math.random() * 6) + 1; // 배열에서 랜덤 1개 선택 const arr = ['홍길동', '김철수', '이영희']; const idx = Math.floor(Math.random() * arr.length); const picked = arr[idx]; // → '홍길동', '김철수', '이영희' 중 하나
📌

Math.floor 란?

소수를 버림해서 정수로 만듭니다.

Math.floor(3.9) → 3
Math.floor(0.7 * 3) → 2

random()*배열길이는 0.0 ~ 2.999...
floor하면 → 0, 1, 2 중 하나

배열 메서드 — push, filter, map

팀원과 업무 목록을 관리하는 배열 조작 메서드

JS let members = ['홍길동', '김철수']; // push: 맨 뒤에 추가 members.push('이영희'); // → ['홍길동','김철수','이영희'] // filter: 조건에 맞는 것만 추출 const without = members.filter( m => m !== '김철수' ); // → ['홍길동','이영희'] // map: 각 항목을 변환 const upper = members.map( m => `👤 ${m}` ); // → ['👤 홍길동','👤 김철수','👤 이영희'] // includes: 포함 여부 확인 members.includes('홍길동'); // → true

팀원 추가/삭제 패턴

JS // 추가 function addMember(name) { if (!name.trim()) return; if (members.includes(name)) { alert('이미 있는 이름!'); return; } members.push(name); render(); } // 삭제 (filter로 제외) function removeMember(name) { members = members.filter( m => m !== name ); render(); }

Fisher-Yates 셔플 — 공정한 섞기

배열을 완전히 무작위로 섞는 알고리즘

단순히 Math.random()으로 정렬하면 특정 순서가 더 자주 나옵니다. Fisher-Yates 알고리즘은 완전히 공정한 셔플을 보장합니다.
function shuffle(arr) { // 원본 배열을 수정하지 않도록 복사 const copy = [...arr]; // 뒤에서부터 랜덤 위치와 교환 for (let i = copy.length - 1; i > 0; i--) { const j = Math.floor(Math.random() * (i + 1)); // i번째와 j번째 위치 교환 [copy[i], copy[j]] = [copy[j], copy[i]]; } return copy; } // 사용 예시 const shuffled = shuffle(['A','B','C','D']); // → ['C','A','D','B'] 같이 섞임
🔄

스프레드 연산자 [...]

[...arr]는 배열을 복사합니다.

원본을 바로 수정하면 나중에 다시 배정할 때 원래 목록이 사라지므로, 복사본을 만들어 셔플합니다.

💾 "고쳤는데 화면이 그대로예요"

⏸️ "Cline이 멈춰서 아무것도 안 해요"

파일을 만들거나 고치기 전에 승인(Approve)을 기다리는 중입니다. Cline 채팅창의 승인 버튼을 눌러주세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.

Number 변환 — 문자를 숫자로

HTML 입력값은 항상 문자열이므로 계산 전에 숫자로 변환해야 합니다

HTML <input>에서 읽은 값은 항상 문자열(string)입니다.
"3" + "5" = "35" (문자열 이어붙기!). 계산하려면 반드시 숫자로 변환해야 합니다.
// 문자열과 숫자의 차이 "3" + "5" // → "35" (문자 이어붙기!) 3 + 5 // → 8 (덧셈!) // 변환 방법 3가지 Number("3.14") // → 3.14 ✅ parseFloat("3.14") // → 3.14 ✅ parseInt("3.14") // → 3 (정수만!) // 숫자인지 확인 isNaN("abc") // → true (숫자 아님) isNaN(3.14) // → false (숫자 맞음) // 0으로 나누면? 10 / 0 // → Infinity
🤔

Number vs parseFloat

Number('') → 0 (빈 문자열 = 0)
parseFloat('') → NaN (못 변환)

계산기에서는 parseFloat이 더 안전합니다.
"3.14abc" → parseFloat → 3.14

🔀

switch 문 — 연산자 분기

if/else if를 여러 번 쓰는 대신 switch로 깔끔하게 분기

JS function calculate(a, op, b) { switch (op) { case '+': return a + b; case '-': return a - b; case '×': return a * b; case '÷': if (b === 0) return '오류'; return a / b; default: return '알 수 없는 연산자'; } } calculate(10, '+', 5); // → 15 calculate(10, '÷', 0); // → '오류'

if/else if vs switch

✅ switch 적합

같은 변수를 여러 값과 비교할 때
연산자(+,-,×,÷) 분기처럼 경우가 명확할 때

✅ if/else 적합

범위 비교(count > 10)할 때
복잡한 조건이 섞일 때

상태 변수 — 계산기의 기억

계산기는 현재 어떤 단계인지를 변수에 기억해야 합니다

상태 변수는 앱이 "지금 어떤 단계인지"를 기억하는 변수입니다.
계산기에서는 첫 번째 숫자, 선택한 연산자, 두 번째 숫자를 따로 기억해야 합니다.
// 계산기의 상태 변수들 let firstNum = null; // 첫 번째 숫자 let operator = null; // 연산자 (+,-,×,÷) let isNewNum = true; // 새 숫자 입력 중? // 연산자 버튼 클릭 시 function clickOp(op) { firstNum = parseFloat(display.value); operator = op; isNewNum = true; // 다음 숫자 입력 대기 } // = 버튼 클릭 시 function clickEqual() { const secondNum = parseFloat(display.value); const result = calculate(firstNum, operator, secondNum); display.value = result; firstNum = result; // 결과를 다음 계산에 사용 }
🔄

계산기 흐름

1️⃣ 숫자 버튼 클릭 → 화면에 표시
2️⃣ 연산자 버튼 클릭 → firstNum 저장
3️⃣ 다시 숫자 버튼 클릭 → 두 번째 숫자
4️⃣ = 클릭 → calculate() 실행
5️⃣ 결과 표시 → firstNum에 저장

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.

코드 핵심 — 숫자 입력 처리

이전 숫자 이어서 입력 vs 새 숫자 시작

JS let currentInput = ''; // 현재 입력 중인 숫자 문자열 let isNewNumber = true; // true: 새 숫자 시작 / false: 이어서 입력 function inputDigit(digit) { if (isNewNumber) { // 연산자 누른 후 → 새 숫자 시작 currentInput = String(digit); isNewNumber = false; } else { // 숫자를 이어서 입력 if (digit === '.' && currentInput.includes('.')) { return; // 소수점 중복 방지 } currentInput += digit; } display.textContent = currentInput; // 화면에 표시 }

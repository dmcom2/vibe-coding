정규식(RegEx) — 패턴으로 텍스트 처리

특정 패턴의 텍스트를 찾고 세고 바꾸는 강력한 도구

정규식(Regular Expression)은 텍스트에서 특정 패턴을 찾는 규칙입니다.
/패턴/플래그 형식으로 작성하며, 글자 수 세기, 검색, 치환에 활용합니다.
패턴의미예시
\s공백(스페이스, 탭, 줄바꿈)단어 구분에 활용
\S공백이 아닌 문자공백 제외 글자 수
\n줄바꿈 문자줄 수 계산
+1개 이상 반복\s+ = 연속 공백
g전체 검색 (플래그)모든 패턴 찾기
const text = " 안녕 하세요 "; // 공백 제거 후 단어 분리 text.trim().split(/\s+/); // → ['안녕', '하세요'] // 공백 아닌 글자 수 세기 text.match(/\S/g)?.length; // → 7 ('안녕하세요') // 줄 수 세기 text.split('\n').length; // 문장 수 (마침표, !, ? 기준) text.match(/[.!?]+/g)?.length || 0;
✂️

split & match — 문자열 분해

split은 나누기, match는 패턴에 맞는 것 찾기

split — 문자열 나누기

JS // 구분자로 나누기 "a,b,c".split(','); // → ['a','b','c'] // 빈 문자열로 나누면 글자 배열 "안녕".split(''); // → ['안','녕'] // 공백(여러 개도)으로 나누기 "안녕 하세요".trim().split(/\s+/); // → ['안녕', '하세요']

match — 패턴 찾기

JS // g 플래그 → 모든 일치 배열 반환 "ab12cd34".match(/\d+/g); // → ['12', '34'] // g 플래그 없음 → 첫 번째만 "ab12cd34".match(/\d+/); // → ['12', index:2, ...] // 일치 없으면 null 반환! "안녕".match(/\d+/g); // → null (→ ?.length || 0 로 처리)

SNS 글자 제한 처리

각 플랫폼의 제한에 맞춰 경고 표시

const SNS_LIMITS = { twitter: { name: '트위터', max: 280 }, kakao: { name: '카카오 카드', max: 100 }, instagram:{ name: '인스타그램', max: 2200 }, email: { name: '이메일 제목', max: 998 } }; function updateSNS(charCount) { Object.entries(SNS_LIMITS).forEach(([key, sns]) => { const el = document.getElementById(key); const pct = Math.min((charCount / sns.max) * 100, 100); const over = charCount > sns.max; el.textContent = `${sns.name}: ${charCount}/${sns.max}`; el.style.color = over ? '#dc2626' : '#059669'; }); }
⚡

input 이벤트 vs change 이벤트

input — 글자를 입력할 때마다 (실시간)
change — 입력 완료 후 포커스 이동 시

실시간 카운터에는 input 이벤트를 사용해야 합니다.

❓

?.length 란?

?.length — Optional Chaining
match() 결과가 null이어도 에러가 나지 않고 undefined를 반환합니다.

🔍 "Go Live 버튼이 안 보여요"

VS Code 왼쪽 확장(Extensions) 메뉴에서 Live Server를 검색해 설치한 뒤, VS Code를 껐다가 다시 켜세요.

💾 "고쳤는데 화면이 그대로예요"

저장(Ctrl+S)이 먼저입니다! 저장하지 않으면 반영되지 않아요. 저장 후 브라우저에서 새로고침(F5) 하세요.

⏸️ "Cline이 멈춰서 아무것도 안 해요"

파일을 만들거나 고치기 전에 승인(Approve)을 기다리는 중입니다. Cline 채팅창의 승인 버튼을 눌러주세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.

코드 핵심 — 통계 계산 함수

하나의 텍스트에서 여러 통계를 한 번에 계산

JS function analyze(text) { return { // 전체 글자 수 total: text.length, // 공백 제외 글자 수 noSpace: text.replace(/\s/g, '').length, // 단어 수 (공백으로 분리, 빈 문자열 제외) words: text.trim() === '' ? 0 : text.trim().split(/\s+/).length, // 문장 수 (마침표/느낌표/물음표 기준) sentences: (text.match(/[.!?]+/g) || []).length, // 줄 수 lines: text === '' ? 0 : text.split('\n').length }; } // textarea 입력 시마다 실행 textarea.addEventListener('input', () => { const stats = analyze(textarea.value); updateDisplay(stats); updateSNS(stats.total); });

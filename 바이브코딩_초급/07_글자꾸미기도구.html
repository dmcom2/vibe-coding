JavaScript로 CSS 바꾸기

element.style 객체를 통해 인라인 CSS 속성을 직접 제어합니다

핵심 원리: HTML 요소는 모두 .style 객체를 가지고 있습니다.
element.style.fontSize = "24px" 처럼 속성을 할당하면 인라인 스타일이 즉시 반영됩니다.
CSS 속성명의 하이픈(-)은 camelCase로 바꿔 씁니다 — font-size → fontSize, font-weight → fontWeight
JavaScript // HTML: <div id="preview">미리보기</div> const preview = document.getElementById('preview'); // CSS 속성명 → camelCase 변환 규칙 preview.style.fontSize = '32px'; // font-size preview.style.color = '#2563eb'; // color preview.style.fontWeight = 'bold'; // font-weight preview.style.fontStyle = 'italic'; // font-style preview.style.background = '#eff6ff'; // background preview.style.textDecoration = 'underline';// text-decoration preview.style.textAlign = 'center'; // text-align // textContent로 표시 내용도 동기화 preview.textContent = document.getElementById('inputText').value;

range 슬라이더

input[type=range]으로 숫자 범위를 드래그로 입력받습니다

HTML + JavaScript <!-- HTML: 슬라이더 + 현재값 표시 --> <label>글자 크기</label> <input type="range" id="sizeRange" min="10" max="80" value="24" oninput="applyStyles()"> <span id="sizeVal">24px</span> // JavaScript: 슬라이더 값 읽어서 적용 function applyStyles() { const val = document.getElementById('sizeRange').value; // "24" (문자열!) document.getElementById('sizeVal').textContent = val + 'px'; document.getElementById('preview').style.fontSize = val + 'px'; } // oninput → 드래그 중에도 실시간 발생 ← 이것을 사용! // onchange → 드래그를 놓았을 때만 발생 ← 덜 즉각적
tip   input.value는 항상 문자열(string)입니다. 숫자 연산이 필요하면 parseInt(val) 또는 +val로 변환하세요. CSS 문자열 조합(val + 'px')에는 변환 없이 바로 사용할 수 있습니다.
🎨

color picker

input[type=color]으로 브라우저 내장 색상 선택기를 불러옵니다

HTML + JavaScript <!-- HTML: 글자 색상 / 배경 색상 피커 --> 글자 색상: <input type="color" id="fontColor" value="#1e293b" oninput="applyStyles()"> 배경 색상: <input type="color" id="bgColor" value="#ffffff" oninput="applyStyles()"> // JavaScript: color picker 값은 #rrggbb 형식 function applyStyles() { const fontColor = document.getElementById('fontColor').value; // "#2563eb" const bgColor = document.getElementById('bgColor').value; // "#f0f9ff" const preview = document.getElementById('preview'); preview.style.color = fontColor; preview.style.background = bgColor; } // 반환값: 항상 소문자 #rrggbb 6자리 hex — CSS에 그대로 사용 가능
tip   color 피커를 클릭하면 OS 기본 색상 선택 창이 열립니다. 팔레트 선택, HEX 직접 입력, 스포이트 도구를 모두 지원합니다.

🔍 "Go Live 버튼이 안 보여요"

VS Code 왼쪽 확장(Extensions) 메뉴에서 Live Server를 검색해 설치한 뒤, VS Code를 껐다가 다시 켜세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.

코드 핵심 — applyStyles()

모든 스타일을 한 함수에서 일괄 읽어 미리보기에 반영합니다

JavaScript function applyStyles() { const preview = document.getElementById('preview'); const text = document.getElementById('inputText').value; const size = document.getElementById('sizeRange').value; const fontColor = document.getElementById('fontColor').value; const bgColor = document.getElementById('bgColor').value; const bold = document.getElementById('chkBold').checked; const italic = document.getElementById('chkItalic').checked; const underline = document.getElementById('chkUnderline').checked; preview.textContent = text || '미리보기 텍스트'; preview.style.fontSize = size + 'px'; preview.style.color = fontColor; preview.style.background = bgColor; preview.style.fontWeight = bold ? 'bold' : 'normal'; preview.style.fontStyle = italic ? 'italic' : 'normal'; preview.style.textDecoration = underline ? 'underline' : 'none'; document.getElementById('sizeVal').textContent = size + 'px'; updateCSSCode(); // CSS 출력 영역도 동기화 }

클립보드 복사

navigator.clipboard API — async/await 비동기 패턴

JavaScript // 현재 스타일로 CSS 코드 문자열 생성 function updateCSSCode() { const s = document.getElementById('preview').style; const css = `font-size: ${s.fontSize}; color: ${s.color}; background: ${s.background}; font-weight: ${s.fontWeight}; font-style: ${s.fontStyle}; text-decoration: ${s.textDecoration};`; document.getElementById('cssOutput').textContent = css; } // async/await로 클립보드에 복사 async function copyCSS() { const text = document.getElementById('cssOutput').textContent; try { await navigator.clipboard.writeText(text); alert('✅ CSS 코드가 복사되었습니다!'); } catch (err) { alert('복사 실패: ' + err.message); } }
주의   navigator.clipboard는 HTTPS 또는 localhost에서만 동작합니다. 로컬 file://에서 복사가 안 될 때는 VS Code의 Live Server 확장을 사용하세요.

JavaScript(JS)란?

HTML이 뼈대라면, JavaScript는 근육입니다

JavaScript — 웹 페이지를 움직이게 만드는 언어입니다.
버튼 클릭, 시간 표시, 데이터 계산 등 모든 동적인 기능을 담당합니다.

역할 비교

HTML

뼈대

페이지 구조, 텍스트, 버튼 배치

CSS

외모

색상, 크기, 폰트, 레이아웃

JS

근육

클릭 반응, 시간 갱신, 계산 처리

이번 실습에서 사용하는 JS

JS // 현재 시간 가져오기 const now = new Date(); // 시/분/초 꺼내기 const h = now.getHours(); const m = now.getMinutes(); const s = now.getSeconds(); // 1초마다 반복 실행 setInterval(updateClock, 1000);
🕐

new Date() — 날짜/시간 정보

현재 날짜와 시간의 모든 정보를 담고 있는 객체

JS const now = new Date(); // 날짜 정보 now.getFullYear() // → 2026 now.getMonth() // → 5 (6월, +1 필요!) now.getDate() // → 30 now.getDay() // → 2 (화요일, 0=일) // 시간 정보 now.getHours() // → 14 now.getMinutes() // → 35 now.getSeconds() // → 22

Date 메서드 정리

메서드반환값주의
getFullYear()연도-
getMonth()0~11+1 해야 실제 월
getDate()1~31-
getDay()0=일,6=토0이 일요일!
getHours()0~2324시간제
🔄

setInterval — 1초마다 반복

지정한 시간 간격으로 함수를 자동 반복 실행

// 1000ms(1초)마다 updateClock 실행 setInterval(updateClock, 1000); function updateClock() { const now = new Date(); const h = String(now.getHours()).padStart(2,'0'); const m = String(now.getMinutes()).padStart(2,'0'); const s = String(now.getSeconds()).padStart(2,'0'); document.getElementById('clock').textContent = `${h}:${m}:${s}`; }
🔢

padStart(2, '0') 란?

숫자가 한 자리일 때 앞에 0을 붙여줍니다

'9'.padStart(2,'0') → '09'
'10'.padStart(2,'0') → '10'

⏰

밀리초 단위

setInterval의 두 번째 인자는 밀리초

1000ms = 1초
500ms = 0.5초
3000ms = 3초

파일 실행 방법

AI가 만든 코드를 VS Code에서 실행하는 4단계

1

폴더 열기

VS Code에서 02_실시간시계 폴더를 엽니다 — 메뉴 → 파일 → 폴더 열기

2

index.html 클릭

왼쪽 탐색기에서 index.html 파일을 클릭해 엽니다

3

Go Live 버튼 클릭

VS Code 화면 우측 하단의 Go Live 버튼 클릭 (Live Server 확장 필요)

4

브라우저에서 확인!

자동으로 브라우저가 열리며 시계가 작동합니다. 코드 수정 후 Ctrl+S로 자동 새로고침

🆘

자주 막히는 문제 & 해결법

여기서 멈추는 게 정상입니다 — 옆 사람도 같은 곳에서 막혀 있어요

🔍 "Go Live 버튼이 안 보여요"

VS Code 왼쪽 확장(Extensions) 메뉴에서 Live Server를 검색해 설치한 뒤, VS Code를 껐다가 다시 켜세요.

💾 "고쳤는데 화면이 그대로예요"

저장(Ctrl+S)이 먼저입니다! 저장하지 않으면 반영되지 않아요. 저장 후 브라우저에서 새로고침(F5) 하세요.

⏸️ "Cline이 멈춰서 아무것도 안 해요"

파일을 만들거나 고치기 전에 승인(Approve)을 기다리는 중입니다. Cline 채팅창의 승인 버튼을 눌러주세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.

수정 미션, 이렇게 진행하세요

바이브 코딩의 진짜 실력은 "고쳐달라고 잘 말하기"에서 나옵니다

1
같은 Cline 대화에서 이어가기

2
미션 문구 붙여넣기

변경 내용 확인하고 승인

4
저장하고 확인

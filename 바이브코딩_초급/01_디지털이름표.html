시작 전 준비물 체크

4가지만 준비되면 바로 시작할 수 있습니다 (이미 되어 있다면 통과!)

💻
VS Code 설치
code.visualstudio.com 에서 무료 다운로드. 코드를 작성·실행하는 우리의 작업대입니다.

바탕화면에 "바이브코딩" 폴더를 만들어두세요. 차시별 결과물을 여기 모읍니다.

오늘 나오는 용어 미리보기

외우지 마세요 — 눈에만 익혀두면 수업이 술술 풀립니다

📌 HTML

웹페이지의 뼈대를 만드는 언어. 문서에서 제목·본문을 나누듯 구조를 잡아줍니다

📌 CSS

뼈대에 색·크기·배치를 입히는 꾸미기 언어. 몸에 옷을 입히는 일과 같아요

📌 태그(Tag)

<h1>처럼 꺾쇠로 내용을 감싸는 표시. 내용물에 붙이는 라벨이에요

📌 브라우저

웹페이지를 보여주는 프로그램(크롬·엣지). 오늘 만든 파일도 여기서 열립니다

📌 index.html

웹사이트의 대문 역할을 하는 기본 파일 이름. 오늘 만들 파일입니다

📌 프롬프트

AI에게 원하는 것을 설명하는 요청 문장. 바이브 코딩의 핵심 도구!

🌱 낯선 단어가 나와도 괜찮습니다. 이 슬라이드로 돌아와서 다시 확인하면 됩니다. 코드는 AI가 쓰고, 우리는 읽고 요청만 하면 돼요!
📖

HTML이란?

웹 페이지를 만드는 기초 언어

HyperText Markup Language — 웹 페이지의 구조를 만드는 언어입니다.
마치 문서에서 제목·본문·글머리 기호를 나누는 것과 같습니다.

HTML 파일의 기본 뼈대

HTML <!DOCTYPE html> <html> <head> <title>페이지 제목</title> </head> <body> <!-- 화면에 보이는 내용 --> <h1>안녕하세요!</h1> </body> </html>
1

<head> 영역

브라우저에 보이지 않는 설정 영역
제목, 스타일 등을 넣습니다

2

<body> 영역

실제로 화면에 보이는 내용을
여기에 작성합니다

🏷️

태그(Tag)란?

HTML을 구성하는 기본 단위 — <태그>내용</태그> 형태

자주 쓰는 태그

<h1>제목</h1> 가장 큰 제목
<p>내용</p> 문단 텍스트
<div>구역</div> 구역(상자) 나누기
<span>글자</span> 인라인 텍스트

이름표에서 사용하는 코드

HTML <!-- 이름표 카드 구조 --> <div class="card"> <!-- 이니셜 아바타 --> <div class="avatar">홍</div> <!-- 이름 (가장 큰 제목) --> <h1 class="name">홍길동</h1> <!-- 직급 · 부서 --> <p class="position">대리 · 마케팅팀</p> </div>
💡 class="name" — 이 태그에 별명을 붙여 CSS에서 꾸밀 때 사용합니다
🎨

CSS — 색상과 크기를 꾸미기

HTML이 뼈대라면, CSS는 옷입니다

카드 스타일 예시

CSS /* 카드 전체 스타일 */ .card { background: #1e3a5f; color: white; border-radius: 20px; padding: 40px; text-align: center; } /* 이름 크게 표시 */ .name { font-size: 2rem; font-weight: bold; }

색상 코드 규칙

#1e3a5f 진한 파랑
#ff6b6b 산호 빨강
#51cf66 밝은 초록

💡 색상 코드가 어렵다면
AI에게 "파스텔 분홍색" 처럼
말로 설명해도 됩니다!

💬

AI에게 말 거는 법 — 프롬프트 3단 구조

말하듯 쓰면 됩니다. 이 순서만 기억하세요

①
무엇을 "디지털 이름표 웹페이지를 만들어줘"
만들 것을 한 문장으로. 첫 줄이 가장 중요합니다
②
내용 "이름: 홍길동, 직급: 대리, 부서: ..."
들어갈 정보를 목록으로 나열합니다
③
조건 "파란색 카드 형태, 이름은 크게, 파일 하나로"
원하는 모양과 형식을 구체적으로 적습니다
❌ 아쉬운 프롬프트
"이름표 만들어줘"
→ AI가 마음대로 만듭니다
✅ 좋은 프롬프트
"파란색 카드 형태로, 이름은 크고 굵게, 이메일에 아이콘 붙여서"
→ 원하는 대로 나옵니다
💡 전문용어를 몰라도 됩니다. "촌스럽지 않게", "회사 분위기에 맞게"처럼 말로 설명해도 AI가 알아듣습니다.

파일 실행 방법

AI가 만든 코드를 VS Code에서 실행하는 4단계

1

폴더 열기

VS Code에서 01_디지털이름표 폴더를 엽니다
메뉴 → 파일 → 폴더 열기

2

index.html 클릭

왼쪽 탐색기에서 index.html 파일을 클릭해 엽니다

3

Go Live 버튼 클릭

VS Code 화면 우측 하단의 Go Live 버튼을 클릭합니다
※ Live Server 확장 프로그램이 필요합니다

4

브라우저에서 확인!

자동으로 브라우저가 열리며 이름표가 표시됩니다
코드를 수정하고 Ctrl+S 저장하면 자동으로 새로고침됩니다

🆘

자주 막히는 문제 & 해결법

여기서 멈추는 게 정상입니다 — 옆 사람도 같은 곳에서 막혀 있어요

🔍 "Go Live 버튼이 안 보여요"

VS Code 왼쪽 확장(Extensions) 메뉴에서 Live Server를 검색해 설치한 뒤, VS Code를 껐다가 다시 켜세요.

💾 "고쳤는데 화면이 그대로예요"

저장(Ctrl+S)이 먼저입니다! 저장하지 않으면 반영되지 않아요. 저장 후 브라우저에서 새로고침(F5) 하세요.

⏸️ "Cline이 멈춰서 아무것도 안 해요"

파일을 만들거나 고치기 전에 승인(Approve)을 기다리는 중입니다. Cline 채팅창의 승인 버튼을 눌러주세요.

🔤 "한글이 깨져 보여요"

AI에게 "한글이 깨져 보여. UTF-8 인코딩으로 고쳐줘"라고 요청하세요. VS Code 오른쪽 아래 인코딩 표시가 UTF-8인지도 확인!

🤖 만능 해결법: 오류 메시지나 이상한 화면을 그대로 복사(캡처)해서 AI에게 "이거 왜 안 돼? 고쳐줘"라고 붙여넣으세요. 대부분 해결됩니다.
🔍

코드 핵심 설명

index.html 파일에서 찾아봐야 할 핵심 부분

① 제목 태그

<h1 class="name">홍길동</h1>
• <h1> — 가장 큰 제목 태그
• class="name" — 이 요소에 'name'이라는 별명 붙이기
• CSS에서 .name { }으로 스타일 적용

② 색상 설정 (CSS)

.card { background: #1e3a5f; /* 배경색 */ color: white; /* 글자색 */ }
• #1e3a5f — 16진수 색상 코드
• # 뒤 6자리 숫자/알파벳으로 색상 표현
• AI에게 색상 변경 요청이 가장 쉬운 방법!

수정 미션, 이렇게 진행하세요

바이브 코딩의 진짜 실력은 "고쳐달라고 잘 말하기"에서 나옵니다

1
같은 Cline 대화에서 이어가기
새 작업(New Task)을 열지 마세요! 코드를 만들어 준 대화에 이어서 요청해야 Cline이 맥락을 기억합니다.
2
미션 문구 붙여넣기
다음 슬라이드의 미션 프롬프트를 복사하거나, 원하는 변화를 말로 설명하세요. 문법·전문용어는 몰라도 됩니다.
3
변경 내용 확인하고 승인
Cline이 파일을 직접 고쳐줍니다. 어떤 부분이 바뀌는지(diff) 훑어보고 승인하세요. 직접 타이핑할 필요가 없어요.
4
저장하고 확인
저장(Ctrl+S) → 브라우저 새로고침(F5) → 바뀐 부분을 확인합니다.
🔑 한 번에 하나씩! 여러 개를 동시에 요청하면 어디가 바뀌었는지 알기 어렵습니다.
😱 망가졌다면? AI에게 "방금 수정을 취소하고 이전 코드로 되돌려줘"라고 하면 됩니다.
